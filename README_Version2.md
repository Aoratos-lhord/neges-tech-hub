# 🚀 Neges Tech Hub

A modern, high-performance blog platform for tech tutorials, campus life stories, and event photography.

## 🎯 Features

- ✅ Fully responsive (mobile-first design)
- ⚡ Fast performance (Vite + React 18)
- 🎨 Beautiful UI (Tailwind CSS)
- 🏷️ Smart filtering (categories & tags)
- 🖼️ Event gallery (masonry layout)
- 📰 Blog system (full article support)
- 🔍 Search functionality
- 📧 Newsletter signup

## 🛠️ Tech Stack

- React 18
- Vite
- TypeScript
- Tailwind CSS
- Zustand (state management)
- React Router (navigation)
- Lucide Icons
