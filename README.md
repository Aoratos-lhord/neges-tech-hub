# 🚀 Neges Tech Hub

A modern, high-performance blog platform for tech tutorials, campus life stories, and event photography.

## Features

✅ **Fully Responsive** - Mobile-first design  
⚡ **Fast Performance** - Built with Vite & React 18  
🎨 **Beautiful UI** - Tailwind CSS with custom branding  
🏷️ **Smart Filtering** - Filter by category and tags  
🖼️ **Event Gallery** - Masonry layout with lightbox  
📰 **Blog System** - Full article support  
🔍 **Search** - Real-time article search  
📧 **Newsletter** - Built-in subscription form  

## Tech Stack

- React 18
- Vite
- TypeScript
- Tailwind CSS
- Zustand (state management)
- React Router (navigation)
- Lucide Icons
